# Install Dependencies
- Backend

    ```
    pip3 install -r requirements.txt
    ```
- Frontend

    [Install](https://nodejs.org/en/) Node.js first, then run
    ```
    cd frontend
    npm install
    ```

# Run
- Backend

    ```
    python3 manage.py runserver
    ```
- Frontend

    ```
    cd frontend
    npm run start
    ```
